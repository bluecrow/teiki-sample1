# TEIKI SAMPLE PROJECT

これは、定期更新ゲームのサンプルプロジェクトです。

## 方針

四本目の剣（4thSword）というゲームを、wwwとAPIで動く、SPA（Single Page Application）として実装していきます。

## ファイルの説明

- keizoku.cgi TSV形式の継続データ
- teiki.php 定期更新ゲーム本体
- www/login.html 宣言画面のひな形
- api/xxx APIを置いていく予定です
